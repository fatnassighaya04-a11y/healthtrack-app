# 🏥 HealthTrack App

Application web de suivi de santé personnelle développée dans le cadre d'un projet agile.

## 📋 Description
HealthTrack permet à l'utilisateur de suivre au quotidien :
- 💧 Consommation d'eau
- 😴 Heures de sommeil
- 🏃 Activité physique
- 😊 Humeur du jour
- 💊 Rappels de médicaments

## 🛠️ Méthodologie
- **Design Thinking** : Empathize → Define → Ideate → Prototype → Test
- **Gestion de projet** : Méthode Agile / Scrum
- **Outil de suivi** : Trello
- **Versioning** : GitHub
